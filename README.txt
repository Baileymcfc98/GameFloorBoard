Immersive Board V3 - Ready-to-upload package

Files:
- index.html: full editable board system
- assets/: placeholder images
- README.txt: this guide

Upload to GitHub Pages:
1. Upload index.html and the assets folder to a GitHub repository.
2. Keep index.html at the root of the repository.
3. Go to Settings > Pages > Deploy from branch > main > root.
4. Use the GitHub Pages link inside Immersive Studio.

Main features:
- Edit Mode for teachers
- Import/delete images and videos
- Export/import board JSON files
- Drag-and-drop tile swapping in Edit Mode
- Small dice button on centre artwork
- Dice animation at bottom-centre
- Keyboard controls for non-interactive floors
- Local browser save

Keyboard controls:
- Space: roll dice
- M: move selected token by last dice roll
- 1, 2, 3, 4: select team token
- Left/Right arrows: move token backward/forward
- E: open/close Edit Mode
- T: open/close Teacher Controls
- Esc: close popups/drawers

Editing workflow:
1. Click Edit Mode or press E.
2. Select a tile from the editor or click a tile on the board.
3. Change title, subtitle, icon, colour, popup text.
4. Import Image/Video to add media.
5. Delete Media to remove media.
6. Drag tiles to swap their positions.
7. Export Board to save a reusable JSON file.
8. Import Board later to reload that version.

Important note about videos:
Imported videos are embedded in the exported JSON file. This is useful for a prototype, but large videos will create very large board files. Use short MP4 clips where possible.
