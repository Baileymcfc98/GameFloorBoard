Immersive Board Game Template - Version 2

Upload this whole folder to a GitHub repository and enable GitHub Pages.

Important:
- The main file must be called index.html.
- Keep the assets folder next to index.html.
- Replace the placeholder SVG images in assets/ with your own JPG, PNG, GIF, SVG, or MP4 files.
- Edit the BOARD_CONFIG section inside index.html to change titles, colours, images, videos, pop-up tasks, tokens, and teacher controls.

Basic GitHub Pages steps:
1. Create a new GitHub repository.
2. Upload everything inside this folder: index.html, README.txt and the assets folder.
3. Go to Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main.
6. Folder: /root.
7. Save.
8. Use the published GitHub Pages link inside Immersive Studio.

Suggested image/video sizes:
- Background: 3840 x 2160 or similar landscape image.
- Centre artwork: 2400 x 1400 or similar.
- Tile images: square or portrait works best.
- Videos: MP4, muted/looping videos work best for tile previews.

Touch use:
- Tap any tile to open its task card.
- Use Teacher Controls to roll dice and move tokens.
- Press Full Screen before starting the experience.
